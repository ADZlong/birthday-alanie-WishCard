# 🎂 Happy Birthday Alanie Fatinie Ramzan

A beautiful, animated birthday wish card website for Alanie Fatinie Ramzan — complete with a gummy-blob-to-cake morphing entrance, floating petals, confetti, and a live wish card editor.

---

## 🌸 Features

- **Animated Gummy → Cake Entrance** — A wobbly gummy blob morphs into a layered birthday cake with flickering candles
- **Sparkle Background** — Hundreds of sparkling particles fill the sky-blue & pink backdrop
- **Spotify Music Embed** — *River Flows In You* (Piano Version) plays directly in the page
- **Wish Card Editor** — Click the button to open a card, type your personal wish, and preview it
- **Preview Mode** — See your wish inside a beautiful framed card with falling confetti 🎉
- **Floating Petals** — Pink blossoms float gently in the background
- **Fully Responsive** — Works on mobile and desktop

## 🚀 How to Publish on GitHub Pages (Step-by-Step)

### Step 1 — Create a GitHub Account
If you don't have one, go to [github.com](https://github.com) and sign up for free.

---

### Step 2 — Create a New Repository

1. Click the **"+"** icon (top right) → **"New repository"**
2. Name it: `birthday-alanie` *(or anything you like)*
3. Make sure it's set to **Public**
4. ✅ Check **"Add a README file"** (optional, we'll replace it)
5. Click **"Create repository"**

---

### Step 3 — Upload Your Files

1. Inside your new repository, click **"Add file"** → **"Upload files"**
2. Drag and drop these three files:
   - `index.html`
   - `style.css`
   - `script.js`
3. Scroll down, write a commit message like `Add birthday card files`
4. Click **"Commit changes"**

---

### Step 4 — Enable GitHub Pages

1. In your repository, click **"Settings"** (top tab bar)
2. Scroll down the left sidebar → click **"Pages"**
3. Under **"Source"**, select:
   - Branch: `main`
   - Folder: `/ (root)`
4. Click **"Save"**
5. Wait ~1–2 minutes…

---

### Step 5 — Get Your Live Link 🎉

After saving, GitHub will show you a green banner with your link:

```
https://YOUR-USERNAME.github.io/birthday-alanie/
```

That's it! Share this link with Alanie! 💕

## ✏️ How to Customize the Wish Text

The wish card has a text area where you can type directly in the browser — no coding needed!

1. Open the website
2. Click **"Open Birthday Card"** button
3. Type your personal message in the text box
4. Click **"Preview Card"** to see the final result with confetti ✨
